📌 IP Assignment: Retinal Vessel Segmentation
📖 Overview

This project implements a classical image processing pipeline for retinal vessel segmentation using the DRIVE dataset.

⚙️ Pipeline
Green channel extraction
CLAHE contrast enhancement
Matched filtering (multi-orientation)
Feature extraction
AdaBoost classification
Morphological post-processing
🧠 Key Insight
Strong performance on major vessels
Thin vessel detection remains challenging
Classical methods depend heavily on feature design
